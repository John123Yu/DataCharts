About

  • Author: John Yu
  • This is a data-science project with visualizations displayed as a web application.
  • Data was aquired from the IRS website: https://www.irs.gov/statistics/soi-tax-stats-individual-income-tax-statistics-zip-code-data-soi.
  • The information is of the county level, not individual level.
  • This project is soley for data exploration. Each zipcode was treated as a row, and clustered using kmeans (6 clusters). Before kmeans, PCA and Standard Scaling was performed. The initial shape of the data is 27681 by 132.
